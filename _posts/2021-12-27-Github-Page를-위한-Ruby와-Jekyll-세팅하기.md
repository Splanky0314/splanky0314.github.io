# Github Page를 위한 Ruby와 Jekyll 세팅하기

## Install Ruby

https://rubyinstaller.org/downloads/
최신버전을 다운받는다.

## Install
터미널에서
```cmd
gem install jekyll bundler
```
를 입력한다.
