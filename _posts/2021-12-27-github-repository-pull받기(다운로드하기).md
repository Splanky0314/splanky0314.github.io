```
git init
git remote add origin https~~
git pull origin main
```

https~부분은 자신의 repository의 링크를 넣어준다.

main은 github repository에서 로컬에 다운받을 브렌치의 이름으로 사용자마다 다르니 확인해보자. (master 등)
