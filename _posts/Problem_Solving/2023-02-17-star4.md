## "2441번 별 찍기 - 4" 정답 코드
```c
#include <stdio.h>
int main() {
	int n;
	scanf("%d", &n);
	for(int i=0; i<n; i++) {
		for(int j=0; j<i; j++) {
			printf(" ");
		}
		for(int j=n; j>i; j--) {
			printf("*");
		}
		printf("\n");
	}
}
```
