## "2556번 별 찍기 - 14" 정답 코드
```c
#include <stdio.h>
int main() {
	int n;
	scanf("%d", &n);
	
	for(int i=0; i<n; i++) {
		for(int j=0; j<n; j++) {
			printf("*");
		}
		printf("\n");
	}
}
```
